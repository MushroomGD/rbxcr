
# Roblox Cursor Replacer 
[![GitHub release (latest by date)](https://img.shields.io/github/v/release/MushroomGD/rbxcr?color=green&label=version)](https://github.com/MushroomGD/rbxcr/releases/latest)

ROBLOX Cursor Replacer (RBXCR) is an application to make changing your ROBLOX cursors easier.


## Running the application

Download [rbxcr.exe](https://github.com/MushroomGD/rbxcr/releases/latest/download/rbxcr.exe) and run it.

### Notes on antiviruses blocking the application

If you do get a warning, it is 100% safe to press Run anyway. Plus, our source code is available for checking.

## Usage

Simply, replace the images of the cursors with your desired ones. Then, when ready to change, press "Apply to Roblox" then confirm. Now, next time when you open Roblox, or leave/join an experience, your cursors will be changed to the new ones!
